<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Node.js
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/ToDo-API-MySQL-Backend.git
   ```
2. Create .env file with environment variables
    ```sh
    API_PORT=3000
    JWT_KEY=super_secret_key
    DB_HOST=localhost
    DB_PORT=3306
    DB_NAME=todos
    DB_USER=root
    DB_PASSWORD=root
    ```
3. Install NPM packages
   ```sh
   npm i
   ```
4. Start API
   ```sh
   npm start
   ```
